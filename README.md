# pimouse_ros

ROSを用いて、ブザーを鳴らします。

## 環境設定
- Raspberry Pi 3 Model B
- Ubuntu 16.04
- ROS (Kinetic Kame)

以下のバッシュスクリプトを用いるとよい。  
[ros_setup_scripts_Ubuntu16.04_server](https://github.com/ryuichiueda/ros_setup_scripts_Ubuntu16.04_server)

## 使い方
端末を立ち上げ以下を実行
```
source ~/.bashrc
```
